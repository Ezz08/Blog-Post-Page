# Blog Post Page

A simple blog post webpage built with HTML. The project focuses on semantic HTML, content organization, accessibility, and proper document structure.

# About the Project

Blog Post Page is a beginner-friendly HTML project that demonstrates how to structure a blog article using semantic elements. It presents headings, paragraphs, lists, images, quotations, and code examples in a clean and meaningful layout.

This project was built to practice writing semantic and accessible HTML while creating a well-structured blog page.

# Features

* Semantic HTML structure
* Organized blog article layout
* Navigation bar
* Multiple content sections
* Headings and paragraphs
* Unordered lists
* Images with captions
* Strong emphasis for important text
* Inline code examples
* Code block demonstration
* Clean and readable document structure

# Getting Started

## Requirements

* Any modern web browser
* Visual Studio Code (recommended)
* Live Server extension (optional)

## Installation

Clone the repository:

```bash
git clone https://github.com/Ezz08/Blog-Post-Page.git
```

Move into the project folder:

```bash
cd Blog-Post-Page
```

# Running the Website

Open `index.html` directly in your browser, or launch the project using Live Server in Visual Studio Code.
